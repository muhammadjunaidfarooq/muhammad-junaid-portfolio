## 🚀 Getting Started

### Prerequisites

* Node.js 18+
* npm

### Installation

#### Clone the repository

```bash
git clone https://github.com/muhammadjunaidfarooq/my-portfolio-website.git
```

#### Navigate to project directory

```bash
cd my-portfolio-website
```

#### Install dependencies

```bash
npm install
```

#### Start development server

```bash
npm run dev
```

#### Open browser

```text
http://localhost:3000
```

---

## 🏗️ Production Build

```bash
npm run build
npm start
```
